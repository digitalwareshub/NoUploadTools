# Open Graph Image

You need to create an `og-image.png` file (1200x630px) for social media sharing.

## Recommended Content:
- NoUploadTools logo
- Tagline: "Privacy-First Browser Utilities"
- Subtitle: "Convert files without uploads"
- Background: Clean, professional design
- Colors: Match your brand (consider the logo colors)

## Tools to create it:
- Canva (free templates available)
- Figma
- Photoshop
- Online OG image generators

Once created, place the file as:
`/public/og-image.png`

The file is already referenced in the metadata configuration.
